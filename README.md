# Dashboard Power BI - Petshop Gatito 🐱📊

## Descrição

Este projeto foi desenvolvido como parte da formação em Power BI da Alura e tem como objetivo aplicar conceitos de Business Intelligence na construção de um dashboard interativo para análise de vendas de um pet shop fictício.

O painel foi projetado para facilitar a tomada de decisão por meio da visualização de indicadores de desempenho (KPIs), filtros dinâmicos e análises gráficas, permitindo explorar informações de forma intuitiva.

## Funcionalidades

- 📈 Faturamento Total
- 🛒 Quantidade de Vendas
- 🐶 Média de Pets por Venda
- 📍 Faturamento por Bairro
- 📅 Evolução do Faturamento ao longo do tempo
- 🔎 Filtros por período, marca e produto

## Tecnologias utilizadas

- Power BI
- Power Query
- DAX
- Modelagem de Dados
- Visualização de Dados (Data Visualization)

## Objetivo

O principal objetivo deste projeto foi consolidar os conhecimentos adquiridos durante o curso, desenvolvendo um dashboard funcional que apoia análises de negócio e a tomada de decisão baseada em dados.

## Aprendizados

Durante o desenvolvimento deste projeto foram aplicados conceitos como:

- Importação e tratamento de dados
- Modelagem relacional
- Criação de medidas utilizando DAX
- Desenvolvimento de KPIs
- Construção de dashboards interativos
- Boas práticas de visualização de dados
